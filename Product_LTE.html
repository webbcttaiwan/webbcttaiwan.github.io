  Overview

  • The LTE RF Integration Test System is designed to test and certify all of the LTE test cases1 for the various LTE products according to the 3GPP TS 36.521-1 chapter 6 and 7.
  • The system uses six configurable test architectures to categorize and speed up the test of the test cases sequentially. Each test-architecture typically contains a common PC server with self–designing software to control the tester, along with any additional commercial equipment such as signal analyzer or analog/vector signal generators, and to execute the required test and measure the result.
  • When any specific test is complete, the customer can read the corresponding test result. The system can set up flexible test configurations (RF path compensation, pass/fail judgment, etc.) through user-friendly windows interface and satisfy different LTE products.
  • The DC Power Supply is optional equipment for automatically test.
1CA and MIMO test cases will be supported in the future.
  Hardware Architecture


The system includes four equipment: LTE Tester, Signal Analyzer, VSG and ASG. It also has a Matrix Box to integrate all paths for automatic test. LTE Tester communicates with EUT. Signal Analyzer measures the spectrum power. VSG and ASG generate modulated and CW interference signals.

  Production Line
For customized testing, LTE production line test system can test LTE mobile phone or LTE wireless network card etc. and verify the related functions. The key test items include power, throuput etc. Production line testing often has specific requirements, for example, rapidly verify production functions, shorten products released time etc. Hyper Taiwan technology can integrate other equipment for customers testing requirement if needed.
