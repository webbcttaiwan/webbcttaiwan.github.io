  關於RTLS
       即時地理定位系統 (RTLS) 是為了即時自動辦識物品和資產，確認個人位置所使用的技術。RTLS的目的是持續追蹤目標位置。
       工業物聯網(IoT)時代的來臨，將讓整體RTLS與資產追蹤市場在2021年成長至目前的三倍，達到150億美元規模。以下為本司方案：
  1. 位置地圖服務| 實時軌跡顯示| 組織架構管理| 電子圖檔警訊| 軟件權限管理
  2. 歷史軌跡存儲與回放軟件
  3. 安防攝像連動與錄製軟件
  4. 區域統計與電子點名軟件
  5. 自動考勤與工時統計軟件
  6. 自動化巡檢任務確認軟件
  7. API /開放平台 :
    位置數據接口| 事件與報表接口| 業務指令下行管道| 傳感數據上行管道
  8. 位置大數據平台 :
    人員熱圖分析| 位置行為分析| 工序效率分析| 供應鏈接路徑分析


  關於WOT
       WoT 是 IoT 的應用層(Application Layer)，並且是使用 Web 技術來打造 APP : IoT + Web-enabled technologies 就是 WoT。
       以 URL 來表示 IoT 裝置；為 IoT 加入 URL 的觀念，就是 Google 提出的 Physical Web 計畫。以下為本司方案：
  1. IoT資料收集系統 (適用跨平台運行)
  2. Modbus TCP 通訊元件測試 (適用跨平台運行)
  3. 線上看板: 線上手動調試信號 (適用跨平台運行)
  4. 生產力4.0 : 智能化生產系統 (適用跨平台運行)
  5. PLC 工業聯網( Modbus rtu/tcp/opc protocol)
  6. QR-code APP :
    在物聯網,設備維護,生產資料收集系統的應用 (適用跨平台運行)
